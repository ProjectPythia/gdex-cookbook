Introduction to Cloud Optimized Data on GDEX

Overview¶

GDEX provides Analysis-Ready Cloud-Optimized (ARCO) versions of select datasets in Zarr or virtualized Zarr (kerchunk, virtualizarr) format. ARCO data is chunked and indexed for fast/parallel access — ideal for large-scale analysis like machine learning or AI workflow.

  1. Check whether a dataset has ARCO data available

  2. Browse and search ARCO variables

  3. Obtain the URL link to the ARCO data

Prerequisites¶

ConceptsImportanceNotes
gdex_introNecessaryDataset IDs and GDEX web UI

  • Time to learn: 10 minutes

Find ARCO data on GDEX¶

To find ARCO data on GDEX, use the dedicated search filter:

  1. Navigate to the GDEX web portal

  2. Press the AI Ready Datasets button under the search bar to display all available Analysis-Ready Cloud-Optimized datasets

  3. One can also press the Zarr Format Datasets button for the all the available zarr format dataset.

Access the ARCO data¶

Once you open a dataset page, use the Data Access tab at the top to view available data links. In the column view, scroll to the desired row (or use the all-data link).

The selected link opens a subgroup view of the dataset. Choose the desired subgroup (for example, a specific time frequency).

You will then reach the final data file page, where you can click the Copy URL button to get the Zarr store access URL (example shown: CESM2 Large Ensemble in Zarr format).

Summary¶

ARCO/Zarr data on GDEX enables fast, chunk-aware remote access without full file downloads. Check availability with the API before assuming ARCO exists for a given dataset.

JRA-3Q Reanalysis Data Workflow

Overview¶

JRA-3Q is the Japan Meteorological Agency’s third-generation reanalysis, providing global atmospheric data from 1947 to present. This notebook demonstrates how to access and analyze JRA-3Q data from GDEX.

  1. Locate JRA-3Q on GDEX and inspect its metadata

  2. Access JRA-3Q data via THREDDS or direct download

  3. Compute a basic diagnostic

  4. Compare with another reanalysis (optional)

Summary¶

JRA-3Q extends the JMA reanalysis record back to 1947 and is accessible on GDEX via THREDDS or direct download. Always verify the dataset ID via the API before building your workflow.

Overview¶

GDEX offers several ways to obtain data files. This notebook introduces the whole file download methods.

The GDEX web UI interface¶

.
This section explains how to navigate from the GDEX landing page to a dataset’s Download page, then use the web interface to discover files efficiently. You will learn where to find core controls such as file path browsing, search, sorting, and filter options.

The portal is useful for both quick manual downloads and preparing larger scripted workflows. In particular, it helps you:

  • narrow file lists by available metadata filters (for example, date range, filename pattern, or directory path),

  • inspect some meta data details before downloading,

  • copy direct file links for HTTP-based automation.

Landing page¶

The GDEX web portal is a centralized place to access all different kinds of services provided by GDEX. To begin your search, you can directly perform a syntax search for the dataset you are interested in.

Dataset Page¶

The dataset page provides an overview of the selected dataset, including:

  • Dataset metadata: Title, description, and temporal coverage

  • Access methods: Available download options (HTTP, Globus, POSIX)

  • Citation: How to cite the dataset

  • Dataset metrics: Statistics on dataset usage

Data Access Tab¶

The Data Access tab at the top of the dataset page displays all available data access methods provided by GDEX. To download files to your local machine, locate the “Web Download (HTTPS)” column on the left. Files are typically organized into groups for easier navigation, though an “All Data Products” view is available to browse the entire dataset.

Dataset Group/Subgroup¶

Because datasets can contain many file types and large file collections, GDEX may provide group/subgroup options to make navigation faster and easier.

Groups and subgroups can be organized by:

  • Temporal frequency (for example, daily, monthly, hourly)

  • Simulation domain (for example, atmosphere, ocean, land)

  • Other dataset-specific categories that improve search efficiency

Dataset download¶

Now we finally get to the data file level. In this interface, you can further narrow down the file list by applying filters. By entering a filter keyword, you can reduce the file list to easily select the files you need to download.

Advanced search¶

For some datasets, if sufficient metadata was captured during archival, the Web Download (HTTPS) interface provides two options:

  • File List

  • File Search

File List provides the standard hierarchical view of groups and files described earlier.
File Search offers an advanced filtering interface that allows detailed selection by metadata attributes, such as variable, time range, and available temporal frequency.

Summary¶

GDEX supports HTTP, Globus, and POSIX access. This notebook demonstrates the GDEX web portal and shows how to use the HTTPS file download service.

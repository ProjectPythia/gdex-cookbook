# GDEX Services

GDEX offers several ways to discover, access, and download data, each suited to different workflows and access levels. The pages in this section introduce each service and link to deeper, hands-on notebooks.

- [TDS](tds_intro.ipynb) — Access datasets remotely via OPeNDAP, NCSS, WMS, and other THREDDS protocols without downloading files.
- [Zarr](zarr_intro.ipynb) — Open Analysis-Ready Cloud-Optimized (ARCO) datasets in Zarr or "Vitualized" Zarr format for fast, chunk-aware remote access.
- [API](api_intro.ipynb) — Use the GDEX REST API to discover datasets, retrieve metadata, and request subsets programmatically.
- [Download](download_intro.ipynb) — Download the entire file to local machine for local access.
- [POSIX](posix_intro.ipynb) — Read GDEX datasets directly from the NCAR HPC filesystem.
- [Globus](globus_intro.ipynb) — Transfer large GDEX datasets to your institution or HPC system at high speed with Globus.
